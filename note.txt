git config --global user.name binhnguyen

Khai báo email

Kiểm tra tên
git config --global user.name

Kiểm tra mail
git config --global user.name

Tạo repository
git init

Add link git
git remote add origin https://github.com/binhnguuyen/bootstrapCapstone.git


Dùng lệnh để add thay đổi lên sàn
git add .

Dùng lệnh để push master lên Git Hub lần đầu
git push -u origin master

Dùng lệnh sau để add những file thay đổi thôi
git add -A

Dùng lệnh sau để kiểm tra những đoạn code đang thay đổi
git status

Dùng lệnh sau để xem những nội dung đc thay đổi 
git commit -m "message"
